## Deploy

```shell
docker pull zfeng/blog
```

## Available Environments

* MAILER_PORT
* MAILER_SECURE
* MAILER_USER
* MAILER_PASS

* ALI_ACCESS_KEY
* ALI_SECRET_KEY
* ALI_REGION
* ALI_BUCKET

* EXTERNAL_URL

## Development.

```shell
yarn
npm run dev
```
